About the ANJY Mailing Lists
The ANJY Mailing Lists are either announce or discussion forums that enbable you to communicate with others of similar interest.

The following lists are currently in use.

  • anjy.announce. This is where you will get all the latest news from ANJY, as well as annoucements of events and interesting pieces of information.
  • anjy.talk. A discussion forum where one can discuss issues and ideas relating to Jewish Youth work and ANJY.
  • anjy.madrichim. Another discussion forum, this time for Madrichim (youth leaders). Ideas can be shared as well as dealing with issues that Madrichim are faced with.
  • anjy.users. There is no subscription to this list, but you are able to send messages to it knowing that it will reach all the ANJY users. A list of these can be found via the ANJY Home page.

How to send a message to a Mailing List
If you want to send a message to any of the ANJY Mailing Lists, then send an email message to [name of list]@ort.org.

For example if you want to announce an event then you would send an email message to anjy.announce@ort.org.
