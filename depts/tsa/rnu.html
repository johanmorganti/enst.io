Récepteurs numériques



English version


Responsable : Marie-Laure Boucheret
E-Mail : bouchere@tlse.enst.fr

Objet :

L'utilisation des techniques numériques se généralise dans les récepteurs et devrait évoluer avec la technologie. Outre les avantages classiques de tels procédés (fiabilité, reproductibilité, tests réduits, etc), elle permet l'obtention de récepteurs reconfigurables pouvant travailler en temps partagé.

Le but de cet enseignement est de permettre le dimensionnement efficace d'un récepteur numérique à partir de spécifications de performances.

Dans le présent module on présente l'aspect théorique en analysant les différentes fonctions implantées dans un récepteur. Au cours du module suivant (SIM), une série de travaux pratiques sur Cossap (logiciel de simulation d'une chaîne de communication numérique) conduit à élaborer un récepteur numérique à partir de spécifications.
