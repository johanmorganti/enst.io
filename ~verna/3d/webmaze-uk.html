WebMaze 1.0
Automatic VRML maze generator


WebMaze automaticaly creates a random maze, according to the parameters to be filled in below. This maze is thrown directly to your 3d browser in VRML format. If your HTML browser doesn't open a 3d viewer like webspace after receiving the document, please look at this page. Otherwise, once the scene loaded, you can save it with the "SAVE AS" command of your 3d viewer.
Note that if the dimensions are not properly specified, WebMaze will return an empty scene.

WebMaze features a special and very quick creation algorithm.
In the mazes generated by this software, there is no unreachable space, and so there is always at least one path between the entrance and the exit. Pretty soon, more configuration parameters to come ...
