Gestion des utilisateurs

  • /etc/passwd

    • mapping uid <-> loginname
    • choisir un nom de login unique
    • appartient à root mais en lecture pour tous (ls -l)
    • mis à jour par /bin/passwd (suid root)
    • contient le mot de passe chiffré (13 caractères)
    • détermine le répertoire d'accueil et le shell initial
    • champ "Gecos" facultatif mais très pratique (e-mail)

      user:aKOxhqX/m=hiP:101:10:Prenom Nom:/home/user:/bin/sh daemon:*:1:1:DAEMON:/:

  • /etc/group

    • mapping nom de groupe <-> membres du groupe
    • utile pour les groupes secondaires

      group:*:10:user1,user2,user3,user4

  • /etc/shadow

    • mapping loginname <-> mot de passe chiffré
    • couplé avec /etc/passwd emputé du champ password

      user:x:101:10:Prenom Nom:/home/user:/bin/sh
      user:aKOxhqX/m=hiP:::::

    • fichier inconsultable mais modifiable par /bin/passwd
