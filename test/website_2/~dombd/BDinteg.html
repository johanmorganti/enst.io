Module Bases de Données


Intégrité logique d'une Base de Données

vérifier la cohérence de données est une tâche ardue quand on dispose de très gros volumes de données (1Go -> 1To) tels que ceux manipulés par les SGBD. Habituellement dispersées dans le code des applications, l'intégrité des données dépend pourtant de leur sémantique et devrait être confiée au SGDB, au cours d'une phase suivant la conception du schéma.

L'objectif de ce cours est de présenter une large variété de contraintes d'intégrité et d'examiner les moyens existants ou potentiels de les vérifier. Là encore, l'approche Bases de Données est intéressante mais elle a ses limites. Les dernières propositions de la norme SQL sont discutées et les systèmes de triggers sont introduits.
