Recherche documentaire

  

 Recherche documentaire pour l'utilisateur final

 Accès à la base de données UNCOVER : cette banque de données bibliographiques couvre l'indexation d' articles de 17000 périodiques et propose leurs sommaires récents en ligne. L'interrogation est gratuite et vous permet de rechercher par sujet, auteur, titre de revue.
A la connexion choisir "Search UnCover Database" puis "Search UnCover Now"

 FirstSearch regroupe plusieurs banques de données bibliographiques indexées, dans tous les domaines. Une interface interactive permet à l'utilisateur de rechercher ses références sans l'aide des documentalistes. La bibliothèque étudie la mise en place de ce service sur le réseau par le biais de "cartes de requêtes"



 Interrogation des bases de données

 Pour vous aider dans vos recherches bibliographiques, les documentalistes procédent avec vous à l'interrogation de bases de données en ligne et/ou à la consultation de CD-ROM
Près de 450 bases spécialisées dans tous les domaines (information scientifique, économique, presse-actualité ...) sont accessibles par les serveurs Dialog, Questel, Cedocar et permettent d'obtenir tout type d'information (bibliographiques, factuelles, texte intégral)

 Accès aux normes IUT-Télécommunications en ligne : un abonnement a été souscrit à ce service pour l'année 1997
