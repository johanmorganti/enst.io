PETIT SUPPLEMENT



Malgré tous les pointeurs que j'ai pu trouvé,aucun n'explique les choses suivantes :
- Tout d'abord,pour mettre un fond sur votre page,utilisez la commande

 < BODY BACKGROUND="xxxxx.xxx"
 TEXT=#nnnnnn LINK=#mmmmmm VLINK=#pppppp >
--> xxxxx.xxx est le nom du fichier image contenant votre fond
--> nnnnnn est le code de la couleur du texte
--> mmmmmm est le code de la couleur de vos liens
--> pppppp est le code de la couleur de vos liens dejà sélectionnés

Remarque : pour trouver le code des couleurs,cliquez ici.

- Pour faire clignoter un texte tapez :

  < BLINK > pour activer le clignotement
  < /BLINK > pour le désactiver
C'est tout pour le moment.
