Architecture Programmable pour le decodage canal



Auteur : Olivier Gay-Bellile

Thèse de l'ENST

Département Electronique

Laboratoire d'accueil : LEP (philips)



Résumé :

Il s'agit d'effectuer la conception d'une architecture programmable pour realiser le decodage canal (demodulation et decodage correcteur d'erreur). L'application visee est la television numerique parce qu'elle utilise differents canaux de transmission (Cable, Satellite, Terrestre) et donc differentes techniques de decodage canal.

Mot-Clés :

architecture programmable, demodulation, television numerique, decodage canal

Title : Programmable architecture for channel decoding

Abstract :

It concerns the conception of a programmable architecture in order to realise channel decoding (demodulation and error correction). The target application is digital television because it uses differents broadcast media (Cable, Satellite, Terrestrial) and so differents channel decoding techniques.
