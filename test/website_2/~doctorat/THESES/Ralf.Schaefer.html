Thèse de l'ENST

Département Informatique

Laboratoire d'accueil : Siemens Corporate Technology, Munich, Germany



Résumé :

Actuellement, MPEG-4, un nouveau standard Multimedia, est en cours de developpement. En meme temps, des applications Java permettent a des systemes de communication d'offrir de nouveaux services et fonctionnalites. De ce point de depart, les buts de cette these sont (1) d'analyser ces deux technologies de base afin d'obtenir une comprehension plus generale des principes derrieres, (2) de synthetiser MPEG-4 et Java afin de fournir des nouvelles fonctionnalites, (3) d'optimiser ce system complet de maniere que des problemes comme la vitesse d'execution sont sont partiellement mieux resolu, (4) de generaliser cet example concrete a une vue systeme plus abstract, base par ex. sur un vue "donnes et control".

Title : Study and Optimisation of Multimedia Applications on Base of Byte Code

Abstract :

Currently, MPEG-4, a new multimedia standard, is being developed. At the same time, Java applications enable communication systems to offer new sevices and functionalities. With this background, the aims of this PhD thesis are to (1) analyze these two basic technologies in order to get a more general understanding of the principles behind, (2) sythesize MPEG-4 and Java in order to provide new functionalities, (3) optimize this overall system in a way that problems like execution time are partially handled in a better way, (4) generalize this concrete example to a more abstract system view, based e.g. on a data-control oriented viewpoint.
