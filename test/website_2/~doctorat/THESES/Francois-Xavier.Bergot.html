Décodage spectral correcteur d'erreurs : dualité débruitage / compression



Auteur : Francois-Xavier Bergot

Thèse de l'ENST

Département Communications



Résumé :

Le décodage correcteur d'erreurs consiste à utiliser la redondance (introduite par le codeur de canal) pour retrouver l'information émise. La compression de source tend à supprimer la redondance naturelle d'un message afin d'en extraire l'information pertinente. Les deux opérations sont duales. La thèse consiste dans un premier temps à étudier comment des outils de codage de canal bien connus peuvent être employés pour comprimer une source. Dans un deuxième temps, on utilisera des codes cycliques construits sur des extensions cyclotomiques de Q pour comprimer des sources continues.

Mot-Clés :

Compression, Error corrrection, codage correcteur d'erreurs, Rate Distortion Theory, théorie de la compression avec perte, cyclic codes, codes cycliques, block codes, codes en bloc

Title : Spectral error decoding : duality between noise suppression and compression

Abstract :

Forward error correction consists in using the redundancy introduced by channel encoder to give a good prediction of the information transmitted. Source compression suppresses the natural source redundancy to extract relevant information. Forward error correction and source compression are dual. First, we will study how we can use the error correcting tools to compress a discrete source. The we will use cyclic codes over cyclotomic extensions of Q to compress continuous sources.
