Etude des réseaux haut-débit pour le partage de mémoire



Auteur : Vincent Habchi

Thèse de l'ENST

Département Informatique



Résumé :

Dans cette thèse nous étudions différents réseaux, tant du point de vue matériel que protocolaire, en vue de leur utilisation dans une machine multiprocesseur à mémoire partagée. Les réseaux standards sont passés en revue, leurs performances mesurées grâce à des cartes de communication spécialement conçues - et nous concluons par une proposition originale de réalisation.

Title : High speed networks for shared memory

Abstract :

In this Ph.D., we explore networks - protocol as well as associated hardware - as support for a shared memory multiprocessor system. Performance of standard networks are mesured with specially designed cards. We conclude by submitting an original proposal and device.
