The Photo3d program


Photo3d is a program to make synthetic images.
These images and these movies have been made with it.

Images are calculated with the ZBuffer algorithm with a phong smoothing. An option allows to calculate anti-aliased images. The calcul of the transparencies doesn't match all cases.

The program is divided in two parts :

    The modeler :
    It's a Macintosh program wich can calculate images too.
    The renderer :
    It's written in C and should run on any UNIX platform (I have compiled it successfully on a DEC ALPHA). It uses xv and mpeg (a program to make mpeg movies from a set of images)
