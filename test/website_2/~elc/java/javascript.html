Nous présentons maintenant le langage JavaScript, qui peut être considéré (en faisant un gros effort d'imagination...) comme un sous(-sous)-langage de Java.

Mais ne soyez pas deçus: JavaScript permet quand même de réaliser des petits programmes et d'effectuer efficacement des contrôles sur les données.
