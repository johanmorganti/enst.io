                     MAINTENANCE DE LA BASE DE DONNEES PHOTOS

Si vous souhaitez metre une nouvelle photographie a la place de l'ancienne dans
la base de donnees photo, envoyez moi un mail a 'sam@enst.fr'.

Format du fichier CONTENTS de la base de donnees:

       nom:prenom:surnoms:chemin_de_la_photo:lisible (y,n ou ?)

ou surnoms est une liste de surnoms separes par des virgules.

Exemple:

         tardieu:samuel:students/promo94cc/tardieu_samuel.gif:y

Pour visualiser les images, executer le programme xv, avec l'option -perfect si
vous etes sur une machine couleur ou en niveaux de gris.
