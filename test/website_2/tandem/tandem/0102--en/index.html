Tips

This activity helps the tandem partners to get to know each other, e.g. after a change of partner. It can also be used as a follow on from 0101.

The most important decisions about the tandem work together (time planning, which language you will communicate in, correction of each other's work) should be taken beforehand. Obviously the questions should be only used as initial guidelines for the partners and then they should talk themselves in as much detail as possible and follow everything up with questions.

Problems

Some learners found a few of the questions too personal to answer openly with their unknown partner. Solution: leave out these questions, change them, or answer them in a light-hearted way (a tandem partnership is not a marriage).

Formats

You can access the activity directly by selecting the simple WWW format (HTML). To be able to access an activity on the more complex RTF format and to be able to print it, you need a word processing programe like MS-Word, WordPerfect or a similar.
