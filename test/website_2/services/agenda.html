Utilisation :

Tableau d'affichage est un utilitaire qui permet afficher des messages sur de differents Tableaux selon sa thématique. en utilisant l'interface graphique de Mosaic.

Tapez sur:

ici pour afficher un message.
Offres pour lire le tableau d'offres.
Demandes pour lire le tableau de demandes.
Suggestions pour lire le tableau de suggestions.
Questions pour lire le tableau de questions.

Mail icon pour envoyer un mail à Joan Biosca.
Info icon pour lire cette page.
Home icon pour retourner à la première page.
