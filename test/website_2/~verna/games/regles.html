Bonjour Jim.

Votre mission, si vous décidez de l'accepter, consistera à nettoyer complètement un champ de mines dans lequel vous allez être parachuté.
Le champ de mines se compose d'un tableau de 25x15 cases. Les mines sont réparties aléatoirement sur les cases (Je ne peux malheureusement pas vous dire combien de mines sont présentes...). Chaque case du tableau est entourée par 8 cases maximum, et 3 cases minimum (les extremités). Chaque case non piegée contient un chiffre correspondant au nombre de mines qui l'entourent (donc entre 0 et 8). Cependant, si la case n'est entourée d'aucune mine, elle est laissee vide (on n'affiche pas le 0).

Sur chaque case, vous avez trois actions possibles:

  • OUVRIR, si vous pensez qu'elle n'est pas piegée. Attention, car l'ouverture d'une case piegée vous fait sauter la tête ! Si vous avez vu juste, la case affichera alors le nombre de mines alentour, ce qui vous donne des informations sur les cases voisines.
  • MARQUER, si vous pensez que la case est piegée.
  • EFFACER,si vous avez marqué une case, mais que vous n'êtes plus très sur qu'elle est piegée. La case sera alors à nouveau considerée comme non explorée.

Votre mission sera accomplie quand vous aurez marqué toutes les cases contenant une mine, et ouvert toutes les autres...

Bonne chance Jim !
