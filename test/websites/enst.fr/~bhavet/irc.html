IRC : International Relay Chat



  • lancer IRC
  • choisir un surnom en faisant /nick surnom . (ce surnom pourra être modifié à tout moment avec /nick nouveau_surnom)-> exemple /nick john
  • rejoindre un canal avec /join #canal. Notez que l'on peut être connecté a plusieurs canaux en même temps -> exemple /join #france
  • tapez un message: toutes les personnes présentes sur le canal le recevront. Inversement le texte qui défile sous vos yeux est tapé par les intervenants dont le nom est au début de la ligne entre "< >"
  • pour partir tapez /bye
  • facile non ?



quelques commandes supplémentaires ...


  • envoyer un message à une seule personne: /msg destinataire message -> exemple /msg bill salut, bill, c'est john

  • savoir qui se cache derrière un surnom: /whois surnom (*) -> exemple /whois bill

  • en savoir plus encore: /ctcp surnom finger -> exemple /ctcp bill finger (*)

  • avoir la liste des connectés à IRC depuis un domaine, quelque soit le canal: /who *.domaine -> exemple: /who *.enst.fr (*)

  • avoir la liste des personnes présentes sur un canal: /name #canal -> exemple /name #france (*)

  • faire apparaitre un message commencant par "* votresurnom texte": /me texte -> exemple: /me a faim affichera "* john a faim"

  • pour ignorer les sollicitations d'un interlocuteur : /ignore surnom all -> exemple /ignore bill all ne fera pas apparaître sur votre écran les messages privés de bill, ses invitations, pings, ...

(*) ces données peuvent être truquées
