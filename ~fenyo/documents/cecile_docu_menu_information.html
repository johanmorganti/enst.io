C.E.C.I.L.E. - Menu Information

Le menu Information comprend trois entrées:
  • Editer
  • Enregistrer
  • Annuler

Editer permet d'éditer le texte dans la partie inférieure de la fenêtre Cecile. Si l'utilisateur vient de cliquer sur un projet, c'est le descriptif de ce dernier qu'il est alors possible d'éditer. Si l'utilisateur vient de cliquer sur un élève, c'est le descriptif de ce dernier qu'il est alors possible d'éditer.

Enregistrer permet de prendre en compte les modifications dues à Editer.

Annuler permet d'annuler les modifications dues à Editer.
