C.E.C.I.L.E. - Exemple de Création de Projet

Pour créer un projet, il faut tout d'abord cliquer sur un département dans la colonne Départements:

Ensuite, cliquez sur l'entrée Créer du menu Projet.

Votre nom se trouve alors rajouté dans la colonne Projets. Vous pouvez rajouter un texte d'explication et modifier le nom du projet, qui est initialement votre nom; pour cela, il vous faut cliquer sur votre nom, puis sélectionner l'entrée Editer du menu Information:

Vous pouvez alors éditer le texte dans la partie inférieure de la fenêtre, sachant que la première ligne contient le nom du projet. Modifiez ainsi la première ligne, qui contient initialement votre nom, pour y mettre un nom adéquate pour votre projet. Les autres lignes peuvent contenir toute information utile concernant votre projet. Pour clore l'édition, sélectionnez Sauvegarder dans le menu Information.
