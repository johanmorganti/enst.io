Two friends of mine and I have written a book on how to connect a LAN to the Internet :

Raccorder son réseau d'entreprise à l'Internet,
Alexandre FENYÖ, Frédéric Le GUERN, Samuel TARDIEU,
Published on february 1997 by Ed. Eyrolles,
513 pages + CD-Rom for Mac/PC/Unix.

Contents (a full description in french is available here):

  • The different steps to connect a LAN to the Internet (modem, RNIS, X.25, leased lines, TCP/IP, PPP...),
  • understanding and configuring the basic services : DNS (Domain Name System), SMTP (Simple Mail Transfer Protocol), NNTP (News servers and clients),
  • The World Wide Web : HTML (HyperText Markup Language), HTTP (HyperText Transfer Protocol), CGI (Common Gateway Interface), installing and configuring servers and proxies,
  • The Multicast Backbone : the concepts, configuring the routing protocols and daemons, installing and using the video and audio multicast applications,
  • Security on the Internet : the Firewall model, building access control lists, software to validate and improve security, cryptography,
  • Many other things I don't have enough time to write here (buy it, you'll love it !).
