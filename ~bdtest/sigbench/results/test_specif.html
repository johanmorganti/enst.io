Test Specification

Test specification suite (INPUT)

From a single test model (for generated data sets, that is, those data models where type = Statistical model) it can be derived into several Test specifications. Each Test specification corresponds to a rectangle generation for each Data model from which one can compute the number of qualifying pairs.

Test specification suite = (Test model suite, {Data sample 1, Data sample 2, Number of pairs retrieved})
where
Selectivity is Number of tuples (pairs) retrieved / (N1 * N2)
N1 - Number of rectangles in Data sample 1
N2 - Number of rectangles in Data sample 2
