Les caractéristiques de JWP

Présentation

JWP est un traitement de texte Japonais/Anglais pour Windows.

Il permet d'entrer du texte en ascii classique, hiragana, katakana et kanji (en utilisant la conversion hiragana/kanji, le dictionnaire ou l'utilitaire de recherche d'un kanji par nombre de traits ou clef). Il est muni du dictionnaire EDICT, qui est un dictionnaire Japonais/Anglais très complet. Vous pouvez imprimer le texte horizontalement ou verticalement et en modifier la taille.

En revanche, JWP ne permet pas encore d'écrire en gras, italique ou souligné, ni d'utiliser des tailles différentes pour diverses parties du document. Toutes ces fonctionnalités sont prévues dans une prochaine version.

Prérequis

JWP fonctionne sur un compatible PC muni de Windows 3.1 ou toute version ultérieure.
Il exitse une version 16 bits pour Windows 3.1, et une version 32 bits pour Windows 95, Windows NT ou Windows 3.1 avec Win32s.
Le minimum recommandé pour des performances acceptables est un 486 à 33 HHz avec 4 Mo de RAM.

Distribution

JWP est un freeware créé par Stephen Chung et est distribué suivant les termes de la GNU General Public License.
Les fichiers que vous trouverez ici correspondent à la version 1.31.
J'ai rapatrié ce logiciel depuis le site ftp.cc.monash.edu.au/pub/nihongo, qui contient de nombreux logiciels relatifs à la langue japonaise.

Fichiers secondaires

Le fichier jwp13doc.zip (715 Ko) contient divers documents : le manuel de JWP, le texte de la GNU General Public License, des notes de copyright et des information sur le codage des textes en japonais.

Le fichier jwp13rec.zip (56 Ko) contient trois petits logiciels de jeux accessibles directement depuis JWP. Si vous n'installez pas ces fichiers, le logiciel fonctionnera correctement, mais vous obtiendrez un message d'erreur si vous essayez de lancer un des jeux.

Comment installer le logiciel

JWP n'est pas accompagné d'un logiciel d'installation automatique  vous devez donc procéder vous même à l'installation.
Pour cela :
  1. Créez un répertoire pour JWP (par exemple c:\jwp).

  2. Placez vous dans ce répertoire et décompressez y tous les fichiers jwp13*.zip.

  3. Déplacez le fichier ctl3dv2.dll (pour la version 16 bits) ou le fichier ctl3d32.dll (pour la version 32 bits) dans le répertoire système de Windows (c:\windows\system).

  4. Vous pouvez maintenant lancer JWP en exécutant le fichier jwp.exe (pour la version 16 bits), ou le fichier jwp32.exe (pour la version 32 bits).

Pour plus de détails sur l'installation, vous pouvez consulter le fichier readme.txt (jwp13prg.zip et jwp13w32.zip) et le fichier jwp32.txt (jwp13w32.txt).
