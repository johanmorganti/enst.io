Diffusion de groupe : IP-multicast

  • Réseau virtuel MBONE (expériences multimédia)

    • MBONE est un réseau virtuel, au dessus de l'Internet, constitué d'ilôts multicast
    • IP multicast (classe D: 224.xx.yy.zz) : adressage de groupe (protocole IGMP)
    • Portée TTL : LAN, site, région, continent, monde

    • Seuil : threshold (propagation si ttl > threshold)
    • Réseau FMBONE en France
    • Routage provisoire soft par daemons mrouted (tunnels)
    • Technique de "tunneling" : encapsulation IP dans IP

  • Applications multicast

    • sd (Session Directory) : annonce des conférences
    • vat (Visual Audio Tool) : audio-conférence
    • wb (White-Board) : paper-board-conférence
    • nv (Network-Video) : vidéo-conférence
    • ivs (Inria Video System) vidéo/audio-conférence
    • telesia (Inria-Aristote) : vidéo/audio-conférence
    • vic (Video Internet Conference) : vidéo/audio-conférence
    • mcm (MultiCast Mosaic) : slide-conférence
    • webcast (MultiCast Mosaic) : groupware-conférence
    • gp (Générateur Poiétique) : art coopératif
