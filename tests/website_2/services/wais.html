Utilisation :

Serveur WAIS est un utilitaire qui facilite les recherches dans la base de données WAIS enst-rapports en utilisant l'interface graphique de Mosaic.

Tapez sur:

Questionner pour poser une question au serveur WAIS sur la base de données enst-rapports.
Envoyer pour savoir comment faire pour intégrer un nouveau document à la base de données enst-rapports.
Pointeur pour accéder http://www.enst.fr/~bioscaj/r à d'autres bases de données.

Mail icon pour envoyer un mail à Joan Biosca.
Info icon pour lire cette page.
Home icon pour retourner à la première page.
