"En quoi le multimédia est-il porteur d'innovation"

L'innovation ne se décrète pas. C'est la maîtrise d'une combinaison subtile de décisions qui impactent les trois dimensions de l'innovation : les structures (les relations de pouvoir au sein des entreprises), les outils (la technologie informatique), les comportements (la psychologie individuelle, et collective).

Technologiquement, le multimédia n'est pas une revolution. C'est une évolution normale et continue, depuis l'invention de l'ordinateur. Economiquement, le multimédia est réalisé par le mariage de l'informatique, des télécoms et de l'audiovisuel.

L'impact sur les comportements est fort. Celui sur les structures est invisible. Le multimédia sera porteur d'innovation, par les déchirements qu'il provoque déjà.
