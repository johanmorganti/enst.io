Application des paradigmes de distribution des langages haut-niveau au cas des machines virtuelles temps-reel embarquees



Auteur : Samuel Tardieu

Thèse de l'ENST

Département Informatique



Résumé :

Le langage Ada 95 dispose de primitives pour distribuer une application originellement monolothique. Le but de cette thèse est de prouver que ces paradigmes sont appliquables dans le cas d'une machine virtuelle temps-réel embarquée, par exemple une machine Java se trouvant à bord d'un robot.

Mot-Clés :

distribution, Ada, robotics, real-time, embedded


Directeur de la thèse : Laurent Pautet

Thèse non Soutenue
Date de soutenance prévue: Fin 1999
Lieu de la soutenance: ENST, Paris


Title : Application of distribution paradigms found in high-level languages applied to embedded real-time virtual machines

Abstract :

The Ada 95 language contains primitives to distribute an application designed to be monolothic. The goal of this PhD is to prove that these paradigms can be applied to a real-time embedded virtual machine, for example a Java VM located in a robot.
