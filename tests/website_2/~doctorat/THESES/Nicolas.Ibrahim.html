Codage pour les communication à étalement de spectre



Auteur : Nicolas Ibrahim

Thèse de l'ENST

Département Signal



Résumé :

L'étalement de spectre des signaux est devenu une méthode bien connue pour un système de communication tolérant la présence de plusieurs utilisateus exploitant les mêmes resources (temps et fréquence). Á la réception, une coopération entre les decodeurs est souhaitable afin d'améliorer les performances du système.

Dans cette thèse nous étudions un schéma de codage (codage canal) qui soit adapté à ce genre d'application. Ainsi nous proposons une méthode d'élimination successive d'interférence basée sur la décision souple obtenu des decodeurs.

Mot-Clés :

Étalement de spectre- Codage canal- Déocdage iteratif- Décodage souple.

Title : Coding for spread spectrum communication

Abstract :

Spread spectrum multiple acces is a popular wirless technology. In its original version, all of the users' signals overlap in time and frequency and cause mutual interference.

Whene encoding the users' signals with a channel code schem, one can improve the performance of the system. The conventional decoder follows a single-user decoding strategy in which the other users' signals appear as noise.

A better strategy is multi-users decoding, where information about multiple users is used to improve decoding of each individual user.

In this thesis, we propose a multi-user encoding/decoding method based on iterative cancellation. A decoding based on soft information is used in order to improve the performances.
