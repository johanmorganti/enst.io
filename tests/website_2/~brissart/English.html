This page deals with my program for image synthesis.

I wrote a raytracing program in C++. It provides a small language that allows the user to describe a scene and then to render it with proper lightning.

This image was made using my own raytracing program. You can notice shadows, reflections, antialiasing, procedural texturing

a few exemples of what it can do :

  1. ellipsoides
  2. anneau
  3. marbre
  4. ocean

I'm currently working on a version of this program which will be much faster and will use Zbuffer algorithm to speed up raytracing. A small demonstration of the current version, showing the fonction of smoothing

  • before smoothing
  • after smoothing
In this new version I do not use CSG anymore. Thus I have been able to use Zbuffer to improve the speed. I also rewrote the octree for secondary rays that allows to compute intersection only with objects that are near the ray. The speed improved a lot and it now only take 1 or 2 minutes to compute a scene with more that 20 000 facets.

  • transparent glass (6 mn on a P166+)
