C.E.C.I.L.E. - Exemple d'Inscription

Pour vous inscrire à un projet, il faut tout d'abord cliquer sur un département dans la colonne Départements, puis sur un projet dans la colonne Projets:

Ensuite, cliquez sur l'entrée Créer du menu Elève.

Votre nom se trouve alors rajouté dans la colonne Elèves. Vous pouvez rajouter un texte de motivation; pour cela, il vous faut cliquer sur votre nom, puis sélectionner l'entrée Editer du menu Information:

Vous pouvez alors éditer le texte dans la partie inférieure de la fenêtre. Pour clore l'édition, sélectionnez Enregistrer dans le menu Information.
