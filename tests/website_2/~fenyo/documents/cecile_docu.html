C.E.C.I.L.E. - Documentation Complète

La version X/Motif de Cecile se trouve installée sur les machines du Département Informatique: /usr/local/bin/cecile

Vous êtes élève, avec Cecile, vous pouvez:

  • Consulter la base de données des projets.
  • Vous inscrire à un projet.

Vous êtes professeur, avec Cecile, vous pouvez:

  • Créer de nouveaux projets.
  • Accepter un ou plusieurs élèves pour un projet.

De plus, des informations sont disponibles sur les fonctions des menus:

  • Réseau
  • Projet
  • Elève
  • Information
