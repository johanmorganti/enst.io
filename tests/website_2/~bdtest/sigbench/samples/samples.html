1st Step: Generic Model choice

The parameters to be specified first are:

  1. Coverage = Sum(area(MBR)) / area(map)
  2. Pr(x) = probability distribution of x coordinate of MBRs lowest left corner
  3. Pr(y) = probability distribution of y coordinate of MBRs lowest left corner
  4. Pr(t) = probability distribution of t slope (between 0 and 90 degrees) of 1st diagonal of MBRs
  5. Pr(a) = probability distribution of a area of MBRs

Please make your generic choices with the buttons bellow. You'll be prompted for secondary parameters to be more specific.

Pr(a) (Coverage and Number of MBR dependent)

2nd Step: Specific Model choice

The parameters to be specified now are:
x = min X coordinate (Western closure of the map)
X = max X coordinate (Eastern closure of the map)
y = min Y coordinate (Southern closure of the map)
Y = max Y coordinate (Northern closure of the map)
(all coordinates in the same INTEGER unit which is up to you to decide)

N = Number of MBR
