Tips

This activity is normally used for partners' first tandem session together. The questions given here make it easier to introduce yourself, or rather to ask your partner questions about him/herself.

The most important decisions about the tandem work together (time planning, which language to communicate in, correction of each other`s work etc.) should be taken beforehand. Obviously these questions should only be used as initial gu idelines by the tandem pair; they should talk themselves in as much detail as possible and should follow up questions with their partner.

Format

You can access the activity directly by selecting the simple WWW format (HTML). To be able to access the activities in the more complex RTF format and to be able to print them, you usually need a word processing progra me like MS-Word, WordPerfect or a similar.
