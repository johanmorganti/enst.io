If you have a SUN or HP workstation and the RPLAY library, you will find these
sound files useful for playing with CrossFire.  Make sure that you have defined
"UseSoundEffects" in "config/crosssite.def" if you want to hear something...

If you don't have the RPLAY library, you can download it from "sounds.sdsu.edu"
(directory "pub/rplay").  Please read the README files in the RPLAY package if
you want to install it.

The most important thing once you have the "rplayd" daemon ready is to build a
configuration file called "rplay.conf".  In this file, you should put the full
pathname to all the sound files that you want to play.  The easiest way to do
this is with this simple command :

   find / -name \*.au -print > rplay.conf

By the way, you may put the sound files anywhere (you don't need to unpack
this archive in one of the crossfire directories) : the "rplayd" program will
be able to find them as long as the path in "rplay.conf" is OK.  Just make
sure that the files are readable from any workstation that will use CrossFire
with the sounds.

These are just a sample of the sounds that you may use with CrossFire.  If you
find better *short* sounds, please let me know.  I'm sure that I have missed
some interesting sound effects...
