Caféine V1.1

Caféine est un editeur de texte écrit en Java pour la programmation Java.

!Il faut absolument posseder le JDK 1.1
Tout d'abord l'archive : editeur.tar.gz
La version en anglais : editeur_eng.tar.gz
Puis l'analyse et un mode d'emploi.
Installation :
(sous unix)
    gzip -d editeur.tar.gz
    tar -oxvf editeur.tar
    Se mettre dans le répertoire Editeur et faire :
    java editeur
(sous windows 95)
    Utiliser WinZip
Liste des fonctionnalités :
    - Chargement /Sauvegarde.
    - Couper/Coller.
    - Gestion d'un curseur de recopie.
    - Gestion d'un projet
    - Compilation
    - Lancement d'applet et d'application
    - Multi-Fenêtre
    - Exécution d'une commande
    - Colorisation syntaxique
    - Repliage
    - Gestion de la tabulation
    - Auto-Indentation
    - Complétion contextuelle
    - Configuration
    - Recherche
    - Aller a une ligne
    - Macros
    - Options de compilation
    - Gestion des erreurs
