Les Applications Java

by Andrej Sobkowski

Cette partie est consacrée à l'étude des bases du langage Java. Vous y trouverez des exemples sur la syntaxe, les propriétés, les Threads, les Exceptions et le traitement des Fichiers.

Cette section n'est pas particulièrement amusante... mais c'est une partie nécessaire pour pouvoir passer à des caractéristiques plus passionnantes.

La syntaxe


L'exemple ExString.java montre les bases du langage, la déclaration de classes, l'héritage, l'overriding et l'overloading.

Passage de paramètres de la ligne de commande


Le fichier ParLigneCommande.java contient le code pour configurer l'exécution d'un programme à partir de la ligne de commande.

La syntaxe acceptée est la suivante:
ParLigneCommande -verbose -cvx -file filename

Les Propriétés


Dans ExProprietes.java on peut trouver un exemple de comment récupérer les propriétés du système et comment en ajouter à partir du fichier Proprietes.txt.

Les Threads


Pour vérifier si le système est fourni de time-slicing, on peut utiliser l'exemple ExThreads.java dans lequel on lance en même temps deux threads: sur des stations Unix, le temps CPU est partagé entre les deux threads.

Les Exceptions


L'exemple ExExceptions.java montre comment traiter les erreurs grâce aux exceptions. En fait, on essait d'ouvrir un fichier qui n'existe pas (vérifier de ne pas avoir dans le répertoire un fichier "abcdef.txt") et d'accéder à un élément d'un tableau avec un index trop élevé.

Dans ExException2.java on crée une nouvelle exception grâce à l'héritage de la classe Exception.

Les Fichiers


Finalement, le fichier ExIOFiles.java contient des informations sur comment lire et écrire un fichier.
